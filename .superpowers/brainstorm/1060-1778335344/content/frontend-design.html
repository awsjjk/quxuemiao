Section 4: Vue 3 前端设计

SPA — 3 种角色 × 5 个核心页面

路由结构

/login → LoginPage
/register → RegisterPage
/dashboard → DashboardPage ← 角色区分：家长/家教/管理员
/demand/:id → DemandDetailPage ← 需求详情 + AI 匹配结果
/order/:id → OrderDetailPage ← 订单详情 + 课程记录
/profile → ProfilePage ← 个人信息 + 编辑
/match/:demand_id → MatchResultPage ← 匹配结果 + 选择家教下单

组件树

App.vue
├── AppLayout.vue ← 全局布局壳
│   ├── NavBar.vue ← 顶部导航 (角色感知)
│   ├── SideMenu.vue ← 侧边菜单 (管理员专用)
│   └── <router-view /> ← 页面内容区
├── views/
│   ├── LoginPage.vue
│   ├── RegisterPage.vue
│   ├── DashboardPage.vue ← 角色分支渲染
│   ├── DemandDetailPage.vue
│   ├── MatchResultPage.vue
│   ├── OrderDetailPage.vue
│   └── ProfilePage.vue
├── components/
│   ├── DemandForm.vue ← 需求发布表单
│   ├── DemandCard.vue ← 需求卡片
│   ├── TutorCard.vue ← 家教推荐卡片
│   ├── OrderCard.vue ← 订单卡片
│   ├── RatingForm.vue ← 评价表单 (星级+文字)
│   └── MatchLoading.vue ← 匹配异步等待动画
├── stores/
│   ├── auth.js ← 用户状态 + JWT
│   └── demand.js ← 需求状态
└── api/
    └── index.js ← Axios 实例 + 拦截器 + 所有请求方法

构建Vite
